### 프로젝트 디렉토리

**mysite(DIR)** : 최상위 루트 디렉토리

- `manage.py`

- **pexels(DIR)** : pexels 프로젝트 디렉토리
    - `__init__.py`
    - `settings.py`
    - `urls.py`
    - `wsgi.py` 
    - `asgi.py`
- **home(DIR)** : home 애플리케이션 디렉토리
    - `__init__.py`
    - `admin.py`
    - `apps.py`
    - migrations(DIR)
        - `__init__.py`
    - `models.py`
    - `tests.py`
    - `views.py`
    - **templates(DIR)** : 애플리케이션의 템플릿 파일 디렉토리
        - home(DIR)
            - `index.html`

- **static(DIR)**

    - css(DIR) : CSS 파일 디렉토리
    - Javascript(DIR) : JS 파일 디렉토리

    - assets(DIR) : 사진 파일 디렉토리
